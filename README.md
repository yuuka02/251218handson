## 🏕️ Äventyr Svenska — スウェーデン語 野外会話トレーナー

Dalslands folkhögskola の äventyr コース(最初の5日間はスマホなしテント泊!)に向けた、
スウェーデン語の**聞き取り練習アプリ**。

### 使い方

`index.html` をブラウザで開くだけ(Chrome / Edge / Safari 推奨)。
音声はブラウザ内蔵のスウェーデン語TTSを使うので、インストール不要・オフラインでも動く。

GitHub Pages を有効にすれば、スマホからもURLで開ける:
リポジトリの **Settings → Pages → Branch を選んで Save** するだけ。

### 機能

- **💬 会話シーン** — テント設営・焚き火料理・ハイキング・天気・体調不良など、テント泊5日間で実際に出てくる8シーンの会話。1行ずつ再生🔊、ゆっくり再生🐢、通し再生に対応。
  - 🙈 **耳だけモード**:スウェーデン語テキストを隠して聞き取りに集中(タップで答え合わせ)
  - 🆘 最重要シーン「**聞き取れなかったとき**」— *Kan du säga det igen?* さえ言えればなんとかなる
- **🎧 聞き取りクイズ** — 音声だけを聞いて意味を4択で当てる全10問。テキストは答えるまで非表示。
- **🃏 フレーズ帳** — 全フレーズ+野外単語(tält, sovsäck, stormkök, myggmedel…)のフラッシュカード。🎙️発音チェック(音声認識)つき。

ヘッダーのスライダーで再生速度を調整できる。最初は🐢ゆっくり、慣れたら等速で。

**Lycka till med äventyret! 🌲**
