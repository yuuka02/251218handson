# アラビア語 単語れんしゅうアプリ 🕌

教科書(第1課〜第7課)の試験範囲の単語・慣用表現を練習できるWebアプリです。
「日本語 → アラビア語」の変換と、アラビア語の綴り(スペル)の練習に特化しています。

## 使い方

### スマホ・PCのブラウザで使う(GitHub Pages)

1. このリポジトリの **Settings → Pages** を開く
2. 「Build and deployment」で **Branch: main / (root)** を選んで Save
3. 数分後に `https://yuuka02.github.io/251218handson/` で使えるようになる
4. スマホのホーム画面に追加しておくといつでも練習できて便利!

### PCでそのまま開く

`index.html` をダブルクリックしてブラウザで開くだけでも動きます(インストール不要)。

## モード

| モード | 内容 |
|---|---|
| 🃏 単語カード | めくって覚えるフラッシュカード(日⇄亜の方向切替あり) |
| ✅ 4択クイズ | ウォームアップ用の選択問題 |
| ⌨️ 綴り(スペル)練習 | 日本語を見て、画面内のアラビア文字キーボードで綴る(メイン機能) |

## 機能

- **課の絞り込み**: 第1課〜第7課をチェックボックスで選択
- **綴り判定は文字のみ**: ハラカット(母音記号)は入力不要。判定後に母音記号付きの正解と、
  どの文字が違ったかの比較(文字タイル)が表示される
- **読みヒント**: カタカナ読みの表示ON/OFF
- **苦手リスト**: 間違えた単語は自動で記録(ブラウザのlocalStorage)。
  「苦手な単語だけ出題」で集中復習できる
- スマホ最適化(モバイルファーストUI)

## ファイル構成

```
index.html   … アプリ本体
style.css    … スタイル
app.js       … クイズロジック・アラビア文字キーボード・判定
data.js      … 単語データ(第1課〜第7課、約100項目)
```

ビルド不要の静的サイト(vanilla JS)なので、`data.js` に単語を足せばそのまま出題に反映されます。
