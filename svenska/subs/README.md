# 毎朝トーク風 字幕ジェネレーター(スウェーデン語ショート動画用)

`make_subs.py` が台本から **ASS**(装飾つき)字幕を生成します。この .ass を緑背景に焼き込んで
クロマキー合成用オーバーレイを作ります(.ass 自体はユーザー納品物ではなく中間ファイル)。

## 生成方法

```bash
python3 make_subs.py
```

出力:
- `20260713_pommes.ass` … ピンク帯ボックス+黒太字+白フチ、装飾つき字幕(焼き込み用)

## フォーマット(スクショ準拠)

| 要素 | 見た目 | 位置 |
|---|---|---|
| 四方フレーム | **白い太枠**(上150px/下200px/左右36px) | 全期間・常時 |
| 通常の台詞テロップ | 上部の**固定長方形**(ピンク地)の中に黒太字+白フチ、1行12文字(全角換算)・**最大2行** | **画面上部** (`TOP_RECT_Y0/H` で調整) |
| タイトル | ピンク大文字 + 黒フチ「Dagligt snack dag ○」(毎日トーク○日目) | 上フレーム内・常時 (`TITLE_TEXT` を毎回書き換え) |
| 「何の日」パネル | **記念日名だけ**を大きく(ピンク地+白フチ枠、白文字+黒フチ) | **顔の下** (`POS_FACE` で調整) |
| 韻(ライミング) | **青 #1E90FF** + 文字拡大 | 台詞内 |
| ダブルミーニング | **赤 #FF3B30** + 文字拡大 | 台詞内 |
| 日付 | ピンク大文字 + 黒フチ「13 juli 2026」(スウェーデン語) | 下フレーム内・常時 |
| タイトル(任意) | `TITLE_TEXT` に文字列を入れると上フレーム内に表示 | 上フレーム内・常時 |

文字サイズ: 基本80 / 仕掛け部分108 / 記念日名96 (`FS_BASE` / `FS_BIG` / `FS_DAY`)
台本のカードは1枚が最大2行に収まるように分割する(超えると生成時にエラーで教えてくれる)。

台本中のタグ: `[R]韻[/R]` / `[W]ダブルミーニング[/W]` / `[DAY]記念日名[/DAY]`

「何の日」パネルは `[DAY]...[/DAY]` の中身だけを表示し、表示区間は
`DAY_START`〜`DAY_END`(該当フレーズの発話区間、実測 13.32〜15.90秒)。
上部字幕としては発表文も全文を通常表示する。

※ 今回の割り当て: frasig・dopp = ダブルミーニング(赤) / smala・skala・gratis・potatis = 韻(青)。
skala は「皮をむく」の掛詞でもありますが、韻の相方として青にしています。

## タイミングについて

`timings.json` に音声(毎日0713.MP3、38.6秒)の実測タイミングが入っています。
ffmpeg の無音検出(silencedetect)で文の区切りを取り、台本10カードに割り当てたものです
(長いポーズ位置がすべて文境界と一致することを確認済み)。

別の動画で使い回すときは `timings.json` を差し替えて `python3 make_subs.py` を再実行。

## プレビュー(焼き込み確認)

```bash
ffmpeg -f lavfi -i "color=c=0xB9AD9C:s=1080x1920:d=60" \
  -vf "subtitles=20260713_pommes.ass" -ss 7 -frames:v 1 preview.png
```

実際の動画に焼き込む場合:

```bash
ffmpeg -i input.mp4 -vf "ass=20260713_pommes.ass" -c:a copy output.mp4
```
